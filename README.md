# Unofficial Sleeping Gods Journal

Use the tool [here](https://alexkazik.github.io/sleeping-journal/).

## Translation

If you want to help with the translation of the website, either update a `*.csv` file in [msg](msg) or copy the
english one and place your language in it.
And then either create a pull request, an issue or contact me (see below).
But be aware that the tool is still changing and entries will (probably) be added/changed/removed.

For translations of the game (all quest names) please contact me (via [email](mailto:sleeping-journal+6437@tx0.eu)
or [BGG](https://boardgamegeek.com/geekmail/compose?touser=txnull)).
